# Wake Creator Dashboard

Web application for creators to upload and manage their training programs.

**Note:** This is the creator-only dashboard. For the client app (PWA), see the main app directory.

## Development

Run the development server:

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

Build for production:

```bash
npm run build
```

The build output will be in the `build/` folder.
