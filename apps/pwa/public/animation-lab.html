Workout Completion

Location: WorkoutCompletionScreen.js — full-screen moment after finishing a workout.
Frequency: Once per workout session (typically 3–6x/week). This is the emotional peak of the entire product. Duration can be long (1–2s+) — user expects to be rewarded.

A — Trophy Burst
🏆

B — Cinematic Phrase
BEAST MODE

C — Gold Confetti
Gold and white confetti particles fall from the top of the screen. Lasts ~2s, then clears. Full celebratory moment.

Skeleton Loading

Location: ProfileScreen header, MainScreen program cards, DailyWorkoutScreen session list.
Frequency: Every screen load / data fetch. Currently inconsistent (some use WakeLoader spinner, some ActivityIndicator). Skeletons feel faster and less jarring.

A — Profile Header
Avatar circle + two text lines for ProfileScreen. Replaces the full-screen WakeLoader spinner. Shimmer runs continuously until data resolves.
B — Program Cards
Two stacked program card skeletons for the MainScreen carousel during load. Heights match actual card dimensions so no layout shift on resolve.
C — Session List Rows
Two session row skeletons for DailyWorkoutScreen. Each row has title + subtitle lines. Opacity decreases for lower rows to simulate depth.
